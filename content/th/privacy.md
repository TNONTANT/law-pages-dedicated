## 4. การเปิดเผยข้อมูลต่อบุคคลที่สาม

เราไม่ขาย แลกเปลี่ยน หรือให้เช่าข้อมูลส่วนบุคคลของคุณแก่บุคคลภายนอก เราเปิดเผยข้อมูลเฉพาะกับผู้ให้บริการ (Data Processor) ดังต่อไปนี้:

| ผู้ให้บริการ | ข้อมูลที่ส่ง | วัตถุประสงค์ | ที่ตั้ง | สัญญาคุ้มครอง |
|------------|-------------|-------------|--------|--------------|
| **Google LLC** (Google Cloud AI — Vertex AI) | รูปถ่ายอาหาร, คำสั่ง AI | วิเคราะห์โภชนาการ | สิงคโปร์ (asia-southeast1) | Google Cloud DPA |
| **Google LLC** (Cloud Run) | Log ระบบ (ไม่มี PII ของ user) | ประมวลผล backend | สิงคโปร์ (asia-southeast1) | Google Cloud DPA |
| **Supabase Inc.** | ข้อมูลผู้ใช้ทั้งหมด | ฐานข้อมูลหลัก | สิงคโปร์ (AWS ap-southeast-1) | Supabase DPA |
| **LINE Corporation** | ข้อความ ผ่าน LINE Messaging API | ส่งข้อความ notification | ญี่ปุ่น/สิงคโปร์ | LINE ToS + Messaging API Terms |

> Google LLC ให้คำมั่นว่าจะไม่นำข้อมูลที่ส่งผ่าน Google Cloud AI ไปใช้ฝึก AI model โดยไม่ได้รับความยินยอมแยกต่างหาก

## 5. การส่งข้อมูลไปยังต่างประเทศ (ม.28)

ข้อมูลของคุณอาจถูกส่งไปยังต่างประเทศตามตารางด้านบน มาตรการคุ้มครอง:

- **สิงคโปร์ (Google Cloud AI / Vertex AI, Google Cloud Run, Supabase):** ครอบคลุมโดย Singapore Personal Data Protection Act 2012 ซึ่ง PDPC ไทยยอมรับว่ามีมาตรฐานเทียบเท่า Google LLC ครอบคลุมเพิ่มเติมโดย Google Cloud Data Processing Addendum (DPA)
- **ญี่ปุ่น/สิงคโปร์ (LINE Corporation):** ครอบคลุมโดย LINE Terms of Service และ APPI (Japan) / Singapore PDPA

## 7. ผลกระทบหากไม่ให้ข้อมูล

- **ไม่ยินยอม cross-border:** ไม่สามารถใช้งาน SnapCal ได้ เนื่องจากการวิเคราะห์รูปอาหารต้องส่งข้อมูลไปยัง Google Cloud AI
- **ไม่ยินยอมข้อมูลสุขภาพ:** ไม่สามารถดูสรุปโภชนาการส่วนตัวหรือคำนวณ TDEE ได้ แต่ยังสามารถส่งรูปอาหารเพื่อดูค่าประมาณทั่วไปได้

---

## 8. ความปลอดภัยของข้อมูล

- การเชื่อมต่อทั้งหมดเข้ารหัสด้วย TLS 1.2+
- ฐานข้อมูล Supabase เข้ารหัสขณะพัก (encryption at rest)
- การเข้าถึงข้อมูลจำกัดเฉพาะผู้ดูแลระบบผ่าน IAM
- ไม่มีการเก็บรหัสผ่านหรือข้อมูลบัตรเครดิต

หากเกิดการละเมิดข้อมูลที่อาจเกิดความเสี่ยงต่อสิทธิ์ของคุณ เราจะแจ้ง PDPC ภายใน 72 ชั่วโมง และแจ้งคุณโดยเร็วที่สุด (ม.37(4))

## 9. การเปลี่ยนแปลงนโยบาย

เราอาจอัพเดทนโยบายนี้เป็นครั้งคราว หากมีการเปลี่ยนแปลงที่กระทบสิทธิ์ของคุณอย่างมีนัยสำคัญ เราจะแจ้งผ่าน LINE ล่วงหน้าอย่างน้อย 30 วัน การใช้บริการต่อเนื่องหลังวันที่มีผลบังคับถือว่าคุณรับทราบการเปลี่ยนแปลง
